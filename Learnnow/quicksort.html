QuickSort

Quick sort uses partition process to divide the input into two parts based on a pivot element where all the elements on the left of the pivot is less than the pivot element and all to the right is greater than it. It is however not necessary for the elements to be in sorted order in both the paritions. But when it is recursively applied to the smallest parition it returns a sorted array.

It can be divided into two parts :-

    Partition
      Choose the last element as pivot
      Choose a pointer which will keep track of all the elements which are less than pivot elements . Lets consider this pointer as i. Initially we will keep its value as one less than starting point because we consider that all the elements as greater than pivot element.
      Iterate from the first element to the second last element of the array.
      Compare each element with pivot element. When getting an element having value less than pivot , we increase the pointer value by 1 and exchange the value so as to bring it to the left of the pivot.
    Call partition with recursive methods
      Call partition method and position the last element at its appropriate position.
      Divide the parition into two parts one to the left of the pivot and other the right of the pivot and sort them recursively with quick sort
Base Step
    Chooss last element as pivot.
    Choose a pointer for elements lesser than pivot.
    If value less than pivot increase the pointer and swap it with the element.
Iterative Steps
    Divide the array into two parts.
    Call the function recursively on left and right partition.
Time complexity analysis
Build max heap for an array of size N= n
No of comparisons for each element=logn
No of elements to be compared=n.
No of steps for heap sort is n+nlogn.
Time complexity of heap sort is O(nlogn).

            print("A is ",A," p is ",p," q is ",q)
            i=p-1
            x=A[q]
            print("i is ",i)
            for j in range(p,q):
              print("For loop")
              if x > A[j]:
                i=i+1
                print("i is ",i)
                print("A[q] ",A[q]," is greater than A[j] ",A[j])
                A[i],A[j]=A[j],A[i]
            print("Returning i is ",i+1)
            A[i+1],A[q]=A[q],A[i+1]
            return i+1
          def quicksort(A,p,r):
            print("A is ",A," p is ",p," r is ",r)
            if p < r:
              q=partition(A,p,r)
              quicksort(A,p,q-1)
              quicksort(A,q+1,r)
