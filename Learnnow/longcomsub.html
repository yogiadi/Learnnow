Longest common subsequence

Base step
    Create a 2D array as per the length of the two arrays.
    If row and column matches , increment the value by1 extracting the value from its diagonally higher element.
    If the element above has higher value then copy over the value from above.
    If the element above has lesser value then the one on the left copy over the value of the left element.
Iterative steps
    Iterate through the 2D array initializing from (0,0)
    While running the print loop , call the function recursively and print the value of the element when the value has been obtained from diagonally higher element.
          
          X=['A','B','C','B','D','A','B']
          Y=['B','D','C','A','B','A']
          def lcs_length(X,Y):
            m=len(X)
            n=len(Y)
            b=[[]]
            c=[[]]
            b=[[0 for j in range(n+1)] for i in range(m+1)]
            c=[[0 for j in range(n+1)] for i in range(m+1)]
            for i in range (0,m):
              for j in range(0,n):
                if X[i] == Y[j]:
                  c[i+1][j+1]=c[i][j]+1
                  b[i+1][j+1]="diag"
                elif c[i][j+1]>=c[i+1][j]:
                  c[i+1][j+1]=c[i][j+1]
                  b[i+1][j+1]="up"
                else :
                  c[i+1][j+1]=c[i+1][j]
                  b[i+1][j+1]="left"
            return c , b

          def print_lcs(b,X,i,j):
            print("i is ",i," and j is ",j)
            if i== 0 or j==0:
              return
            if b[i][j]=="diag":
              print_lcs(b,X,i-1,j-1)
              print(X[i-1])
            elif b[i][j]=="up":
              print_lcs(b,X,i-1,j)
            else:
              print_lcs(b,X,i,j-1)
