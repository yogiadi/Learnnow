Shortest Path

Base step
    The destination vertex of the edge needs should have a key less than equal to the sum of weight of the edge and the value of the key of source edge.
    This relaxation (above process) is done on the basis or priority queue similar to the prim algorithm .
    Matrix is created with all the edges along with the weights.
    Shortest path between two vertex i , j will be shortest path between i and k and edge between j and k.
    To take into account all the edges it should be repeated for V-1 as to have a simple path (not cyclic) between any two vertices for a graph V -1 should be the most number of edges. 
          
def extend_shortest_path(L,W):
  n = len(L)
  L_dash=[[math.inf for i in range(n)] for i in range(n)]
  for i in range(n):
    for j in range(n):
      for k in range(n):
        L_dash[i][j]=min(L_dash[i][j] , L[i][k]+W[k][j])
  return L_dash
def slow_all_pairs_shortest_path(W):
  n=len(W)
  base_L = W
  for m in range(1,n):
    L_temp=[[math.inf for i in range(n)] for i in range(n)]
    L_temp=extend_shortest_path(base_L ,W)
    base_L =L_temp
  return base_L 

def faster_all_pairs_shortest_path(W):
  n=len(W)
  base_L=W
  for m in range(1,n):
    L_temp=[[math.inf for i in range(n)] for i in range(n)]
    L_temp=extend_shortest_path(base_L ,base_L)
    base_L=L_temp
    m=2*m
  return base_L
