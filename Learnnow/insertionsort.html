Insertion Sort

Insertion sort is a sorting method where an array is divided into two parts , one which is already sorted and other which needs to be sorted and the elements are added in such a way that the resultant array is sorted.
Base Step

    Store element in a variable.
    Compare with elements in sorted array.
    Shift element if given element is less than elements.
Iterative step
    Iterate over all the elements starting from second position.
For example consider below array
7 2 9 10 34 27 5 4
Lets divide the array into two parts one which is sorted and other as not sorted.
7 - Sorted
2 9 10 34 27 5 4 - Unsorted
Now insert 2 in the sorted array such that the first part remains sorted.
2 7 - Sorted
9 10 34 27 5 4 - Unsorted
Similarly inserting other elements will result in
2 7 9 - Sorted
10 34 27 5 4 - Unsorted
2 7 9 10- Sorted
34 27 5 4 - Unsorted
2 7 9 10 34 - Sorted
27 5 4 - Unsorted
2 7 9 10 27 34- Sorted
5 4 - Unsorted
2 5 7 9 10 27 34- Sorted
4 - Unsorted
2 4 5 7 9 10 27 34- Sorted Lets build pseudo code for insertion sort considering an array A with length as n :-
Step1. Divide array into two parts sorted and unsorted. We will divide the array into a sorted array of first element and the rest of the elements as unsorted array.So iteration will start from second element to the last element.
for i=2 to n
Step2. Comparison will start from i-1 position , so we can assign a variable called j.
j=i-1
Step3. Store the value of A[i] in a temporary variable temp.
temp=A[i]
Step4. A[i] element will keep moving to left until j =0 or A[i] is greater than A[j]
while A[j]>key and j > 0
Step5. To keep moving the elements shift jth element to j+1 element.
A[j+1]=A[j]
Step6. jth pointer needs to move to left after each comparison
j=j-1
Step7.After loop and comparison key should be established at (j+1)th element.
A[j+1]=key
Hence pseudo code for insertion sort is
for i=2 to n
    j=i-1
    temp=A[i]
    while A[j]>key and j > 0
        A[j+1]=A[j]
        j=j-1
    A[j+1]=key
